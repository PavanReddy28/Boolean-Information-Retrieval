# IR-Assignment

## Stopword Removal

```
$ python preprocess.py <word1> <word2> <word3>
```

## Normalization - Stemming or Lemmatization
Stemming
```
$ python preprocess.py -stem <word1> <word2> <word3>
```
Lemmatization
```
$ python preprocess.py -lemmatize <word1> <word2> <word3>
```
